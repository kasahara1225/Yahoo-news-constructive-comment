# データ形式

各コメントファイル（CSV形式）には以下の行が含まれます：

| 行名         | 説明                              |
|--------------|-----------------------------------|
| 記事タイトル   | コメントが属する記事のタイトル         |
| 本文         | 記事本文                            |
| コメント       | コメントの本文                      |
| 建設的人数     | 建設的と判定した人数（0〜3の整数値）    |
| 非建設的人数   | 非建設的と判定した人数（0〜3の整数値）  |
| 強度         | 建設的割合を「無・弱・中・強」で分類    |
| 度合い       | 建設的割合（数値形式, 0.0〜1.0）      |
