# データ形式

各コメントファイル（CSV形式）には「コメント」と「記事」というシートが存在します．

**シート名：記事**
| 行名         | 説明                              |
|--------------|-----------------------------------|
| 記事タイトル   | 記事のタイトル       　　　　　　  |
| 本文         | 記事本文                           |

**シート名：コメント**
コメントとコメントに付与された建設的程度が記載さ入れています．
| 行名         | 説明                              |
|--------------|-----------------------------------|
| コメント       | コメントの本文                      |
| 建設的     | 建設的と判定した人数（0〜3の整数値）    |
| 非建設的   | 非建設的と判定した人数（0〜3の整数値）  |
| 強度         | 建設的割合を「無・弱・中・強」に割り振り  |
| 度合い       | 建設的割合（数値形式, 0.0〜1.0）      |
